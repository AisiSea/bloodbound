# 物品图标目录

请将物品图标放在本目录下，文件名使用小写英文并与物品类型一一对应：

- `quill.png`：`QUILL`
- `sword.png`：`SWORD`
- `shield.png`：`SHIELD`
- `staff.png`：`STAFF`
- `fan.png`：`FAN`
- `curse.png`：`CURSE`

前端当前会从以下路径读取：

- `/images/seat-icons/item/quill.png`
- `/images/seat-icons/item/sword.png`
- `/images/seat-icons/item/shield.png`
- `/images/seat-icons/item/staff.png`
- `/images/seat-icons/item/fan.png`
- `/images/seat-icons/item/curse.png`

在图片尚未提供前，界面会继续保留物品文字说明。
